逝爱 - 使用说明
===================

快速开始：
1. 确保已安装：
   - Python 3.10+ (https://www.python.org/downloads/)
   - Node.js 18+ (https://nodejs.org/)

2. 安装依赖（首次运行）：
   npm install
   cd backend
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   cd ..

3. 启动应用：
   - 终端1（后端）：cd backend && venv\Scripts\activate && python -m uvicorn app:app --host 0.0.0.0 --port 18000 --reload
   - 终端2（前端）：npm run dev:frontend

4. 打开浏览器访问：http://localhost:5173
