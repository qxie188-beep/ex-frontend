# 逝爱 - 打包说明

## 快速开始（推荐）

直接双击 `快速启动.bat` 即可启动应用！

## 方式一：使用批处理文件（最简单）

### 文件说明

- **`快速启动.bat`** - 推荐使用，自动检查环境并启动
- **`启动逝爱.bat`** - 详细版本，显示更多信息

### 使用步骤

1. 确保已安装：
   - Python 3.10+ (https://www.python.org/downloads/)
   - Node.js 18+ (https://nodejs.org/)

2. 双击 `快速启动.bat`

3. 等待依赖安装完成（首次运行需要）

4. 在浏览器中打开 http://localhost:5173

## 方式二：打包成EXE文件

### 1. 安装打包工具

```bash
# 安装 PyInstaller（用于打包Python后端）
pip install pyinstaller

# 安装 pkg（用于打包Node.js前端，可选）
npm install -g pkg
```

### 2. 单独打包后端

在 `backend` 目录下创建打包脚本：

```python
# backend/build_backend.py
import PyInstaller.__main__
import os

PyInstaller.__main__.run([
    '--name=逝爱后端',
    '--onefile',
    '--windowed',
    '--add-data=app:app',
    'app/__init__.py',
])
```

### 3. 使用 Electron 打包整个应用（推荐）

创建 Electron 项目来包装前端和后端：

```bash
# 在项目根目录
npm install --save-dev electron electron-builder
```

创建 `electron/main.js`：

```javascript
const { app, BrowserWindow } = require('electron');
const { spawn } = require('child_process');
const path = require('path');

let mainWindow;
let backendProcess;

function createWindow() {
  mainWindow = new BrowserWindow({
    width: 1200,
    height: 800,
    webPreferences: {
      nodeIntegration: false
    }
  });

  // 启动后端
  backendProcess = spawn('python', ['-m', 'uvicorn', 'app:app', '--host', '0.0.0.0', '--port', '18000'], {
    cwd: path.join(__dirname, '..', 'backend')
  });

  // 加载前端（开发模式或生产模式）
  mainWindow.loadURL('http://localhost:5173');
}

app.whenReady().then(createWindow);

app.on('window-all-closed', () => {
  if (process.platform !== 'darwin') {
    app.quit();
    if (backendProcess) {
      backendProcess.kill();
    }
  }
});
```

### 4. 使用 Inno Setup 创建安装程序（Windows）

创建 `setup.iss` 文件：

```iss
[Setup]

AppVersion=1.0.0
DefaultDirName={pf}\逝爱
DefaultGroupName=逝爱
OutputBaseFilename=逝爱安装程序
Compression=lzma
SolidCompression=yes

[Files]
Source: "你的项目路径\*"; DestDir: "{app}"; Flags: ignoreversion recursesubdirs createallsubdirs

[Icons]
Name: "{group}\逝爱"; Filename: "{app}\快速启动.bat"
Name: "{commondesktop}\逝爱"; Filename: "{app}\快速启动.bat"

[Run]
Filename: "{app}\快速启动.bat"; Description: "启动逝爱"; Flags: nowait postinstall skipifsilent
```

## 方式三：使用 Docker（跨平台）

创建 `Dockerfile`：

```dockerfile
# 多阶段构建
FROM node:18-alpine AS frontend
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build

FROM python:3.11-slim AS backend
WORKDIR /app
COPY backend/requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY backend/ .

FROM python:3.11-slim
WORKDIR /app
COPY --from=frontend /app/dist ./frontend
COPY --from=backend /app ./backend

EXPOSE 5173 18000

CMD ["sh", "-c", "cd backend && python -m uvicorn app:app --host 0.0.0.0 --port 18000 & cd frontend && python -m http.server 5173"]
```

## 推荐方案

对于Windows用户，**推荐使用 `快速启动.bat`**，因为：

1. ✅ 最简单，无需额外工具
2. ✅ 自动检查和安装依赖
3. ✅ 跨版本兼容性好
4. ✅ 易于调试和更新

如果需要分发，可以使用 **Inno Setup** 创建安装程序，将整个项目打包成一个 `.exe` 安装包。

## 常见问题

**Q: 批处理文件闪退怎么办？**
A: 右键选择"编辑"，在最后添加 `pause` 查看错误信息。

**Q: 如何停止服务？**
A: 在命令行窗口按 `Ctrl+C`。

**Q: 数据保存在哪里？**
A: 所有记忆保存在 `backend/app/exes/` 目录中。
